What are you listening too?

August 21, 2015 · 2 min · admin

This is a brief half-year report on some of the new music I Have heard and enjoyed over the last 6 months (January - July). some are famous, some are not, some you have heard of, some you will  not. anyway it is interesting to try to gather them together for your information and delight. If you wish you can let me know what you have been listening to in the comments section. Against Illusions and Reality by Parks, Squares and Alleys - File under Dreampop. All Are Saved by Fred Thomas  - File under Indie Rock. A Bad Wind Blows in my Heart by Bill Ryder Jones - File under Indie Folk. Broken Temples by Kevin Max - File under Pop. Carrie & Lowell by Sufjan Stevens - File under Indie Folk. Currents by Tame Impala - File under Rock. Eas by Iain Morrison - File under Folk. Escape From Evil by Lower Dens - File under Indie Rock. Girls In Peacetime Want to Dance by Belle & Sebastian - File Under Indie Pop. Goon by Tobias Jesso Jr - File under Indie Pop. Growing Up is for Trees by I’m From Barcelona - File under Indie Pop. Lucid Dreaming by Say Lou Lou - File under Dreampop. Natalie Prass by Natalie Prass - file under Indie Pop. Plague - The Eastern Sea - File under Rock. The Scene Between by The Go Team - File under Electronic. Short Movie by Laura Marling - File under Indie. Sleeping Operator by The Barr Brothers - File under Country. Sometimes I Sit and Think and Sometimes I Just Sit by Courtney Barnett - File under Rock. Stranger Kings by Stranger Kings - File under Dreampop. The Waterfall by My Morning Jacket - File under Rock. What a Terrible World, What a Beautiful World by The Decemberists - File under Indie Pop.
